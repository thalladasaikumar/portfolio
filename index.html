Action Rules Extraction Project

Java AWT applets for UI to Extract Action Rules. An action rule is a rule extracted from an information system that describes a possible transition of objects from one state to another with respect to a distinguished attribute called a decision attribute.

Action rules can be constructed from certain parts of classification rules, but the construction turns out to be expensive.

Action rules can be seen as logical terms, describing, knowledge about possible actions associated with objects which are hidden in a decision system.
